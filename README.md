# "Giorgio Ulani - Nurtrition Shop" 

Aplikacja bazodanowowa napisana pod bazę danych Oracle. Klient bazy jest tworzony w Javie.

## Stage 1

### Built with
- Toad Data Modeler 5.2
- Oracle Database 11g Release 2
- SQL Developer 18.1
